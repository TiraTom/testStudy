## お題
1~100までの数をプリント。
３の倍数はFizz
５の倍数はBuzz
３と５の両方の倍数はFizzBuzzとプリントする。

## まずは課題を分割！表現を揃える

## 課題を書き出す
[ ] 数を文字列に変換する
  [x] 1を渡すと文字列"1"を返す　（「数を文字列に変換する」だと抽象的すぎた）：仮実装
  [x] 2を渡すと文字列"2"を返す：三角測量
[x] 3の倍数の時は数の代わりにFizzに変換する -> 仮実装＆実装
[ ] 5の倍数の時は数の代わりにBuzzに変換する
[ ] 3と5の量の方の倍数の時は数の代わりにFizzBuzzに変換する

[ ]1~nまで
[ ]1~100まで　は後回し（１〜nまでできれば十分なはず）
[ ] プリントする　は後回し　テスト容易性が低く、viewのテストでもあるので旨味が少ない
